cursor: alias;

O cursor indica um alias de algo que será criado

cursor: all-scroll;

O cursor indica que algo pode ser rolado em qualquer direção

cursor: auto;

Padrão. O navegador define um cursor

cursor: cell;

O cursor indica que uma célula (ou conjunto de células) pode ser selecionada

cursor: col-resize;

O cursor indica que a coluna pode ser redimensionada horizontalmente

cursor: context-menu;

O cursor indica que um menu de contexto está disponível

cursor: copy;

O cursor indica que algo deve ser copiado

cursor: crosshair;

O cursor é renderizado como uma cruz

cursor: default;

O cursor padrão

cursor: e-resize;

O cursor indica que uma borda de uma caixa deve ser movida para a direita (leste)

cursor: ew-resize;

Indica um cursor de redimensionamento bidirecional

cursor: grab;

O cursor indica que algo pode ser agarrado

cursor: grabbing;

O cursor indica que algo pode ser agarrado

cursor: help;

O cursor indica que a ajuda está disponível

cursor: move;

O cursor indica que algo deve ser movido

cursor: n-resize;

O cursor indica que uma borda de uma caixa deve ser movida para cima (norte)

cursor: ne-resize;

O cursor indica que uma borda de uma caixa deve ser movida para cima e para a direita (norte/leste)

cursor: nesw-resize;

Indica um cursor de redimensionamento bidirecional

cursor: ns-resize;

Indica um cursor de redimensionamento bidirecional

cursor: nw-resize;

O cursor indica que uma borda de uma caixa deve ser movida para cima e para a esquerda (norte/oeste)

cursor: nwse-resize;

Indica um cursor de redimensionamento bidirecional

cursor: no-drop;

O cursor indica que o item arrastado não pode ser solto aqui

cursor: none;

Nenhum cursor é renderizado para o elemento

cursor: not-allowed;

O cursor indica que a ação solicitada não será executada

cursor: pointer;

O cursor é um ponteiro e indica um link

cursor: progress;

O cursor indica que o programa está ocupado (em andamento)

cursor: row-resize;

O cursor indica que a linha pode ser redimensionada verticalmente

cursor: s-resize;

O cursor indica que uma borda de uma caixa deve ser movida para baixo (sul)

cursor: se-resize;

O cursor indica que uma borda de uma caixa deve ser movida para baixo e para a direita (sul/leste)

cursor: sw-resize;

O cursor indica que uma borda de uma caixa deve ser movida para baixo e para a esquerda (sul/oeste)

cursor: text;

O cursor indica o texto que pode ser selecionado

cursor: url(scroll.png),auto;

Uma lista separada por vírgulas de URLs para cursores personalizados. Nota: Sempre especifique um cursor genérico no final da lista, caso nenhum dos cursores definidos pela URL possa ser usado.
A imagem deve ter no máximo 32x32 pixels ou poderá não ser renderizada.

cursor: w-resize;

O cursor indica que uma borda de uma caixa deve ser movida para a esquerda (oeste)

cursor: wait;

O cursor indica que o programa está ocupado

cursor: zoom-in;

O cursor indica que algo pode ser ampliado

cursor: zoom-out;

O cursor indica que algo pode ser reduzido
